# 🌱 Plant Sensor App

Android-приложение на **Kotlin** для мониторинга состояния растений через Bluetooth (BLE).
Подключается к беспроводному датчику и показывает в реальном времени: влажность почвы, температуру, освещённость и заряд батареи.

---

## Скриншоты

| Экран поиска | Дашборд |
|---|---|
| Сканирует BLE-устройства рядом | 4 карточки с показателями растения |

---

## Возможности

| Функция | Описание |
|---|---|
| 📡 **BLE Scan** | Поиск датчиков растений по Bluetooth Low Energy |
| 🔗 **Подключение** | Автоматическое подключение и чтение данных через GATT |
| 💧 **Влажность** | Процент влажности почвы + статус (Сухо / Оптимально / Перелив) |
| 🌡️ **Температура** | Температура в °C + статус (Холодно / Оптимально / Жарко) |
| ☀️ **Освещённость** | Уровень света в % + статус (Темно / Хорошо / Ярко) |
| 🔋 **Батарея** | Заряд датчика в % + статус (Критично / Заряжен) |
| 🎮 **Demo Mode** | Работает без реального датчика — симулирует данные |

## Как запустить

### Шаг 1 — Установи Android Studio
Скачай с официального сайта: [developer.android.com/studio](https://developer.android.com/studio)

> Android Studio — это программа для разработки Android-приложений. Она включает всё нужное: редактор кода, компилятор, эмулятор телефона.

### Шаг 2 — Открой проект
```
File → Open → выбери папку PlantSensorApp
```
Android Studio сам скачает все нужные библиотеки (это займёт несколько минут).

### Шаг 3 — Запусти приложение
- **На телефоне:** подключи Android-телефон через USB, нажми ▶️ Run
- **На эмуляторе:** создай виртуальное устройство через `Device Manager` и нажми ▶️ Run

### Шаг 4 — Попробуй Demo Mode
Нажми кнопку **"Try Demo Mode"** — приложение покажет живые данные без реального датчика.

## BLE Протокол (для разработчиков)

Приложение ожидает датчик со следующими UUID характеристиками:

| Параметр | UUID |
|---|---|
| **Сервис** | `0000181A-0000-1000-8000-00805F9B34FB` |
| **Влажность** | `00002A6F-0000-1000-8000-00805F9B34FB` |
| **Температура** | `00002A6E-0000-1000-8000-00805F9B34FB` |
| **Свет** | `00002A77-0000-1000-8000-00805F9B34FB` |
| **Батарея** | `00002A19-0000-1000-8000-00805F9B34FB` |

Формат данных: `uint16`, little-endian, значение × 100 (например, `5050` = 50.50%)

## Разрешения приложения

Приложение запрашивает разрешения при первом запуске:

- **Android 12+:** `BLUETOOTH_SCAN`, `BLUETOOTH_CONNECT`
- **Android 8–11:** `ACCESS_FINE_LOCATION` (требуется для BLE сканирования)

## Как это работает (для новичков)

```
Датчик в горшке  →  Bluetooth  →  Телефон  →  Приложение
  (ESP32/Arduino)      (BLE)      (Android)    (этот проект)
```

1. **Датчик** измеряет влажность, температуру и т.д. и отправляет данные по Bluetooth
2. **BleManager.kt** сканирует устройства рядом и подключается к датчику
3. **SensorViewModel.kt** получает данные и хранит их
4. **DashboardActivity.kt** берёт данные и показывает красивые карточки на экране
5. Данные обновляются каждую секунду автоматически
